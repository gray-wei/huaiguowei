# Guowei Huai Homepage

This repository publishes Guowei Huai's academic homepage at:

<https://gray-wei.github.io/huaiguowei/>

The live site is currently a small static homepage generated by Node.js scripts. It is **not** built with Jekyll in the deployment path.

## Current Setup

- `scripts/build-preview.mjs` builds the static site into `_site/`.
- `scripts/serve-preview.mjs` serves `_site/` locally.
- `tests/preview-build.test.mjs` verifies the generated homepage content and required assets.
- `.github/workflows/deploy.yml` runs `npm run test:preview` and deploys `_site/` to GitHub Pages.

Some al-folio/Jekyll files may still exist in the repository from an earlier version, but the current GitHub Pages deployment does not use `bundle exec jekyll build`.

## Local Preview

```bash
npm run test:preview
npm run preview:serve
```

Then open:

<http://127.0.0.1:4173/>

## Deployment

Push changes to `master` on GitHub. The `Deploy site` workflow builds the static preview and publishes the generated `_site/` directory to the `gh-pages` branch.

> Note: Do not edit `gh-pages` directly. It is generated by GitHub Actions.
