# Guowei Huai Homepage

This is a first-stage academic homepage scaffold based on [al-folio](https://github.com/alshedivat/al-folio).

## Current Scope

- Basic identity and research profile
- Publications, projects, CV, and news pages
- Placeholder project pages for the first review pass
- English and Chinese CV PDF downloads

Project images, videos, final publication metadata, and social/profile links are intentionally left for the next phase.

## Local Preview

```bash
bundle install
bundle exec jekyll serve
```

Then open `http://127.0.0.1:4000`.

The bundled GitHub Actions deployment uses Ruby `3.3.5`; local Ruby should match that version.
