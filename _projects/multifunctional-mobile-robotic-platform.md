This project develops a multifunctional mobile robotic platform for home-oriented service robotics.
The system integrates legged locomotion, robot-arm coordination, target tracking and picking, dexterous hand teleoperation, tactile sensing, and multimodal data collection into a unified mobile manipulation workflow.
