This system connects human motion capture, dexterous hand retargeting, arm teleoperation, tactile sensing, and demonstration collection for coordinated robot arm-hand control.
It supports Vision Pro keypoints, Manus glove signals, GELLO leader-follower control, 3D mouse control, and several robot hands including Inspire Hand, Leap Hand, XHand, and Sharpa.

**Authors:** Guowei HUAI, [Linyi HUANG](https://hly-123.github.io/).

**Current status:** Teleoperation System; media assets and system architecture details will be added in the next phase.

**Keywords:** teleoperation, retargeting, multimodal data, tactile sensing, imitation learning.
