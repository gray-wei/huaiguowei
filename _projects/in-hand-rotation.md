This project explores goal-conditioned in-hand teleoperation for multi-axis object rotation.
The current framework uses a two-stage reinforcement learning pipeline in IsaacLab: first learning force-closure grasp poses, then training a multi-axis in-hand rotation policy initialized from those grasp states.
